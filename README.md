A java english dictionary library based on RiTa WordNet

### Maven Dependency for Consumers
Add to your `pom.xml`
```xml
<dependency>
	<groupId>me.shib.java.lib</groupId>
	<artifactId>diction</artifactId>
	<version>0.1.1</version>
</dependency>
```
