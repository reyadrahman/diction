A java dictionary service based on RiTa WordNet library

### Maven Dependency for Consumers
Add to your `pom.xml`
```xml
<dependency>
	<groupId>me.shib.java.lib</groupId>
	<artifactId>diction</artifactId>
	<version>0.1</version>
</dependency>
```
